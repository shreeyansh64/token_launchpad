# Solana Token Launchpad (React)

A simple React-based launchpad to create and manage Solana SPL tokens.

## Features
- Create new SPL tokens
- Mint and distribute tokens
- Connect with Phantom or Solana wallets
- Simple, responsive UI

## Tech Stack
- React.js
- Solana Web3.js
- SPL Token Library
- Tailwind CSS / Styled Components (optional)

## Getting Started
```bash
git clone github.com/shreeyansh64/token_launchpad.git
cd token_launchpad
npm install
npm run dev
```
